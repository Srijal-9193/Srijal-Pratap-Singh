# AI Student Performance Prediction

## Project Description
This project uses Machine Learning algorithms to predict student performance based on study hours and attendance.

## Technologies Used
- Python
- Scikit-Learn
- Pandas
- Matplotlib

## Dataset
The dataset contains student study hours and exam scores.

## Steps in Project
1. Load dataset
2. Split training and testing data
3. Train machine learning model
4. Predict results
5. Evaluate model accuracy

## How to Run
1. Install required libraries
2. Run the Python file in VS Code
3. Check prediction output
